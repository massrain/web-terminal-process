#### Framework
- nextjs 12.1.6
- react 18.2.0
- typescript 4.7.3

#### Commands
- First install without node_modules:
`yarn`

- Run development server:
`yarn dev`

- Install a package:
`yarn add axios`

- Install a dev-dependency package:
`yarn add --dev axios`
